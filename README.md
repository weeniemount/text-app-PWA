# Text PWA App

Just a text editor for Chrome OS and Chrome and the Web. originally from [text-app](https://github.com/GoogleChromeLabs/text-app) made by GoogleChromeLabs

# how to run

you can download the source code and open the index.html, or spin up a local server and open it from there.

the app works as a PWA so you can install it and have it integrated in ChromeOS or whatever
